# go_tree
一个用 Go 编写的命令行工具，用于扫描目录结构并生成可视化的 HTML 文件树报告。支持文件大小统计、百分比展示和交互式折叠功能。
 我来帮你重新排版这个 README，让它更美观、专业：

## 特性

| 特性 | 说明 |
|:---:|:---|
|**大小统计** | 自动计算目录和文件大小，显示占总空间的百分比 |
|**可视化展示** | 生成 HTML 报告，支持目录折叠/展开 |
|**高亮显示** | 占用空间超过 10% 的文件/目录自动标红加粗 |
|**排序** | 按大小降序排列，快速定位大文件 |


## 使用方法

```bash
./main <根目录路径> <输出HTML文件>
```

## 示例

```bash
# 扫描当前目录
./main . report.html

# 扫描特定目录
./main /home/user/projects ~/Desktop/project_tree.html

```
